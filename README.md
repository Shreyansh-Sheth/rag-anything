# RAG with scalable infrastructure to accommodate 10000 of users simultaneously

Best Features:

- Streaming Chat
- Citations
- Worker Indexing

Technology Stack:

- Language: **Typescript**
- Backend: **NodeJS**
- Queue : **BullMQ**
- Database: **MongoDB**
- Vector Database: **ChromaDB**
- Frontend: **NextJS**
